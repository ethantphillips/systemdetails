# Changelog

## 1.0.0

- Initial `mysystem` release.
- Renamed CLI/package from `mysys` concept to `mysystem`.
- Added detailed system reporting for OS, CPU, memory, GPU, storage, motherboard, BIOS, battery, displays, developer tools, package managers, runtime, optional network info, and optional environment info.
- Added text, Markdown, and JSON output.
- Added privacy-safe defaults and full mode.
- Added JavaScript library exports.
